---

### 发现2：为什么震荡市（2022）也失败？

**预期：** 震荡市应该适合均值回归
**实际：** -97.32%亏损

**可能原因：**

#### 1. 波动率过高
```
震荡 ≠ 均值回归的理想环境

理想震荡:
  价格在4,000-5,000箱体中波动
  多次回归均值
  上下边界清晰

2022年实际:
  价格从4,800涨到7,300 (+52%)
  然后跌回5,400 (-26%)
  波动幅度过大，超出止损范围
```

#### 2. 趋势过滤器误判
```
2022年R² = 0.09 (极低)
趋势过滤器认为"无明显趋势"
允许交易 → 实际存在隐藏趋势
```

#### 3. 止损设置问题
```
在大幅波动中:
- 入场阈值1.5可能触发太晚
- 止损倍数1.5可能太宽松
- 50天持仓太长，亏损累积
```

---

### 发现3：即使在牛市/震荡市，均值回归也失败

**传统认知：**
- 均值回归适合震荡市
- 趋势策略适合趋势市

**2020-2023年实证：**
- 牛市（2021）：-99.51%亏损
- 震荡市（2022）：-97.32%亏损
- 弱趋势（2023）：-99.76%亏损
- 熊市（2020）：0%收益（空仓）

**结论：**
TA期货的特性可能完全不适合均值回归策略。

## 四、根本原因分析

### 为什么TA期货不适合均值回归？

#### 1. 市场微观结构
```
TA（PTA期货）特点:
- 受原油价格影响大
- 供需关系变化快
- 政策影响明显
- 趋势性强，均值回归窗口期极短
```

#### 2. 波动率特征
```
日内和日线波动都很大:
- 2022年最高7,364，最低4,833
- 波动幅度达52%
- 超出均值回归的止损范围
```

#### 3. 趋势持续性
```
即使震荡，也有趋势方向:
- 2021年整体上涨31%
- 2022年冲高回落
- 2023年继续上涨9%
"均值"不断上移，难以回归
```
